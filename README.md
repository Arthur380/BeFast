# iRacing Telemetry Dashboards

A collection of **real-time dashboards for iRacing** designed to display important race information while you drive.

These dashboards appear as **lightweight overlay widgets** and provide quick access to useful telemetry such as lap times, relative gaps, race position, and session information.

The goal of this project is to provide **clear, minimal and easy-to-read information** so you can stay focused on racing.

---

# Overview

This application displays several **dashboards (widgets)** that update in real time while you are racing in iRacing.

The dashboards are designed to:

* show the **most important race data**
* remain **easy to read at a glance**
* avoid unnecessary information
* help you make better decisions during a race

The widgets can be positioned on your screen so they fit your preferred racing layout.

# Available Dashboards

## Relative

The **Relative dashboard** shows the cars around you on track.

It typically displays:

* The cars ahead and behind you
* Time gap between your car and others car
* Driver names
* Race / Qualify / Practice position
* Class position
* iRating / SR
* Status : PIT / OUT or L12 (if the car has pit at the lap 12)
* Number of incidents in the session
* Session type (practice, qualifying, race)
* Remaining time or laps

This dashboard is very useful to quickly understand your **race situation** and manage battles with nearby drivers.

---

## Standing

This dashboard provides general information about the current session, such as:

* Bests/last laps of each car on every classes
* Driver names and numbers
* Class position of each car
* Number of cars in every classes
* Number of positions gained
* Gap with the first car
* iRating
* Status : PIT / OUT or L12 (if the car has pit at the lap 12)

It helps you stay aware of the **overall race progress**.

---

## Lap Times

The **Lap Time dashboard** focuses on performance data.

It usually includes:

* Last lap time
* Best lap time
* Time difference compared to previous laps

This helps you track your **pace and consistency** during a session.

---

## Panel

This dashboard recreate a simplified car **panel**, it displays

* Gear
* Delta with the personal best lap
* Speed (km/h)
* RPM leds
* RPM number
* Current position
* Track temperature
* Air temperature
* Wind velocity
* Track wetness
* Time in game
* Pit limiter ON

# Purpose of the Project

The main goal of this project is to create dashboards that are:

* **simple**
* **fast to read**
* **useful during intense racing situations**

Instead of displaying large amounts of telemetry data, the dashboards focus only on the **information that actually matters during a race**.
